## 多个tab标签

``` html   
<div class="tab">
 <ul class="tab-header">
  <li><a>公司1</a></li>
  <li class="now"><a>公司2</a></li>
  <li><a>非常长的标签名称</a></li>
 </ul>
 <div class=“tab-content now">
   你好1
   <br>
   <br>
</div>
<div class="tab-content">
你好2
  <br>
  <br>       
 </div>
</div>
```

## 单个tab标签

``` html   
<div class="tab">
 <ul class="tab-header">
   <li><a>公司1</a></li>
 </ul>
<div class="now tab-content">
   你好1
   <br>
   <br>
 </div>
</div>
  
```
