# uikit

# Build

```sh
node build
node server
```
# 兼容性

* ie: >=6.0.0
* chrome: >=1.0.

`v0.91.0`,`v0.33.9`兼容IE6，以上版本兼容IE9+。

## 0.36.9 版本 2016-05-26

* 更新 theme 模块：添加注释。theme 模块用于存放设计师的配色，定义整个UI的风格。
* 添加 progress 模块
* 添加 tip 模块
* 添加 sizzle 模块
* 添加 calendar 模块：日历模块，提供获取当前年、月、日列表。当前日期前几天，后几天
* 添加 util 模块
* 添加 dateUtil 模块
* 添加 placeholder 模块。使得IE6789可以兼容placeholder属性
